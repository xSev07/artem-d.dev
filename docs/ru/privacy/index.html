Политика конфиденциальности

Дата последнего обновления: 03.03.2026

Разработчик: A.D.A - Artem D

Контактный адрес: contact@artem-d.dev

1. Общие положения

Настоящая Политика конфиденциальности описывает порядок обработки данных при использовании мобильного приложения. Приложение разработано по принципу локального хранения данных (offline-first). Пользовательские данные не передаются разработчику и не отправляются на внешние серверы.

2. Какие данные обрабатываются

2.1 Данные, вводимые пользователем

Пользователь самостоятельно вводит в приложение:

  • финансовые записи (транзакции)
  • категории и теги
  • наименования счетов и другие пользовательские параметры

Эти данные сохраняются исключительно на устройстве пользователя, не передаются разработчику, не копируются на серверы и не передаются третьим лицам. Разработчик не имеет технической возможности доступа к этим данным.

2.2 Технические данные платформы

При установке и использовании приложения платформа Google Play может автоматически обрабатывать технические данные, такие как информация об устройстве, версия операционной системы, диагностические данные и сведения о сбоях.

Обработка таких данных осуществляется сервисами Google в соответствии с их собственной политикой конфиденциальности. Разработчик не получает персональные данные пользователя от Google.

2.3 Встроенные покупки

Приложение может использовать встроенные покупки, обрабатываемые через Google Play. Обработка платежей осуществляется инфраструктурой Google. Разработчик не получает данные банковских карт, не хранит платёжную информацию и не имеет доступа к финансовым реквизитам пользователя.

3. Хранение данных

Все пользовательские данные хранятся исключительно локально на устройстве пользователя в базе данных приложения. Удаление приложения приводит к удалению локально сохранённых данных, если пользователь предварительно не создал резервную копию. Разработчик не хранит копии пользовательских данных и не имеет к ним доступа.

4. Передача данных третьим лицам

Разработчик не осуществляет передачу пользовательских данных третьим лицам. Пользовательские данные не передаются за пределы устройства пользователя.

5. Резервное копирование

Если приложение предоставляет возможность создания резервной копии, файл резервной копии создаётся пользователем и сохраняется по его выбору. Разработчик не получает доступ к резервным копиям.

6. Права пользователя

Пользователь может:

  • удалить все данные, удалив приложение
  • отказаться от использования встроенных покупок
  • связаться с разработчиком по вопросам конфиденциальности

7. Безопасность

Приложение не передаёт пользовательские финансовые данные через интернет. Пользователь самостоятельно отвечает за безопасность своего устройства и созданных резервных копий.

8. Изменения политики

Обновлённая версия политики публикуется на официальной странице приложения.

9. Контактная информация

По вопросам, связанным с конфиденциальностью: contact@artem-d.dev
