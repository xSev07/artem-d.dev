Privacy Policy

Last updated: 03.03.2026

Developer: A.D.A - Artem D

1. General Provisions

This Privacy Policy describes how data is processed when using the mobile application. The app is designed with local data storage in mind (offline-first). User data is not transferred to the developer and is not sent to external servers.

2. What data is processed

2.1 Data entered by the user

The user enters the following into the app:

  • financial records (transactions)
  • categories and tags
  • account names and other user settings

This data is stored exclusively on the user's device, is not transferred to the developer, is not copied to servers, and is not shared with third parties. The developer has no technical ability to access this data.

2.2 Platform technical data

When installing and using the app, the Google Play platform may automatically process technical data such as device information, operating system version, diagnostic data, and crash reports.

Such data is processed by Google services in accordance with their own privacy policy. The developer does not receive the user's personal data from Google.

2.3 In-app purchases

The app may use in-app purchases processed through Google Play. Payment processing is handled by Google's infrastructure. The developer does not receive bank card data, does not store payment information, and does not have access to the user's financial details.

3. Data storage

All user data is stored exclusively locally on the user's device in the app's database. Deleting the app results in deletion of locally stored data unless the user has created a backup beforehand. The developer does not store copies of user data and has no access to it.

4. Transfer of data to third parties

The developer does not transfer user data to third parties. User data is not transferred outside the user's device.

5. Backups

If the app provides the ability to create a backup, the backup file is created by the user and saved at their discretion. The developer does not have access to backups.

6. User rights

The user may:

  • delete all data by deleting the app
  • opt out of using in-app purchases
  • contact the developer regarding privacy

7. Security

The app does not transmit the user's financial data over the internet. The user is responsible for the security of their device and any backups created.

8. Policy changes

An updated version of the policy is published on the app's official page.

9. Contact information

For privacy-related inquiries: contact@artem-d.dev
